# Guidelines for Designing Your Project PowerPoint Presentation

## Project Assignment

### 1. Presentation Format

Please prepare a **PowerPoint presentation** describing the:

- Design
- Development
- Implementation

that you conducted and developed.

The presentation should contain approximately **10–12 slides**.

---

## 2. Required Sections

### A. Introduction

Present the project including:

- Goal
- Problem
- Topic

---

### B. Design Documentation

#### 1. General Description

- Steps and design
- Main instances/modules

#### 2. Design & Architecture

Present your solution as a **block scheme**.

#### 3. Targets and Main Features

#### 4. Design/Development Concepts

#### 5. Design/Development Steps

---

### C. Implementation / Development Documentation

- Present the design & architecture of your system
- Draw and explain the main modules
- Describe functionality and flow

Include:

- At least **2 detailed example cases**

---

### D. Summary

- Reintroduce your project topic
- Explain how you solved the problem
- Emphasize how you used:
  - IoT broker
  - Device
  - Protocol
- Present summary and conclusions

---

### E. Bibliography and Sources

Make sure sources are:

- Properly formatted
- Clearly written

---

### F. Appendix

If there is supporting material, include:

- Data
- Code
- Applications
- Additional files
